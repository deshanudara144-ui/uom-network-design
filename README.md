# University of Moratuwa LAN & Backbone Network Design

This project presents a scalable, reliable, and cost-effective network design for the **University of Moratuwa (UoM)**, including:

- A high-speed backbone network
- A structured LAN for the ENTC building

Designed and simulated using Cisco Packet Tracer.

---

## 🚀 Features
- Hierarchical 3-layer network (Core, Distribution, Access)
- VLAN segmentation for efficient traffic management
- OSPF routing for dynamic path selection
- Spanning Tree Protocol (STP) for loop prevention
- Dual-stack IPv4 & IPv6 support (SLAAC)

---

## 🏗️ Network Architecture

### 🔹 Backbone Network
![Backbone](images/uo.png)

- Hybrid mesh-ring topology

- High-performance Layer 3 switching

### 🔹 ENTC LAN
![ENTC](images/entc.png)

- VLAN-based segmentation
- OSPF for inter-VLAN routing
- Wireless and wired connectivity
